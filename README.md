# 📁 Python File Manager CLI

A simple and beginner-friendly command-line application built using Python that allows users to perform basic file operations such as **Create, Read, Update, and Delete (CRUD)**.

---

## 🚀 Features

* 📄 Create new files
* 📖 Read existing files
* ✏️ Update file content (overwrite / append / rename)
* 🗑️ Delete files
* 📂 Display all files and folders in the directory

---

## 🛠️ Technologies Used

* Python
* `pathlib` module
* `os` module

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/python-file-manager-cli.git
```

2. Navigate to the project folder:

```
cd python-file-manager-cli
```

3. Run the program:

```
python main.py
```

---

## 💡 How It Works

The program provides a menu-driven interface:

```
Press 1 for creating a file  
Press 2 for reading a file  
Press 3 for updating a file  
Press 4 for deleting a file  
```

Users can interactively choose operations and manage files easily.
